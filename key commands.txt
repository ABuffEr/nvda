Key assignments

Note: some commands use the insert key, only the numpad insert is used at this point in ttime.

General 
Quit NVDA: insert+q
Show NVDA window: insert+n (get rid of it with alt+f4)
Time and date: insert+F12 
Toggle speech on and off: insert+s 
increase rate: insert+pageUp
decrease rate: insert+pageDown
toggle speaking of typed characters: insert+2
toggle speaking of typed words: insert+3
toggle speaking of command keys: insert+4
toggle speaking of punctuation: insert+p
toggle speaking of object under mouse: insert+m

Review commands (Edit fields / documents / consoles):
beginning of line: shift+numpad1
previous character: numpad1
current character: numpad2
next character: numpad3
end of line: shift+numpad3
previous word: numpad4
current word: numpad5
next word: numpad6
top line: shift+numpad7
previous line: numpad7
current line: numpad8
next line: numpad9
bottom line: shift+numpad9
Move to caret: numpad subtract
sayAll: numpadAdd

Caret commands (edit fields):
Read format info: insert+f
say all: insert+downArrow

Object navigation (moving only moves the navigator, not the focus)
read currentObject: insert+numpad5
move to and read previous object: insert+numpad4
move to and read next object: insert+numpad6
move to and read parent object: insert+numpad8
move to and read first child object: insert+numpad2
Do object default action: insert+numpadReturn
read current object and ansesters: insert+numpadAdd
Move mouse to navigator object: insert+numpadDivide
move Navigator object to mouse: insert+numpadMultiply

Virtual buffers:
toggle virtual buffer pass-through mode: insert+space
